# 📊 Sales Analysis - Superstore Dataset

## 💼 Problema de Negócio

Analisar dados de vendas para entender padrões de receita, lucro e desempenho por região e categoria.

## 🎯 Objetivo

Aplicar análise de dados com Python para extrair insights básicos e desenvolver habilidades práticas em Data Science.

## 📚 Contexto

Sou estudante de Data Science pela Alura e este projeto faz parte da minha jornada de aprendizado prático.

Meu foco é desenvolver habilidades com dados reais e evoluir para atuar profissionalmente na área.

## 🤖 Uso de Inteligência Artificial

Utilizei Inteligência Artificial como ferramenta de apoio para:

* organização do projeto
* revisão das análises
* melhoria da clareza dos resultados

Todo o conteúdo foi estudado e compreendido como parte do processo de aprendizado.

## 🛠️ Tecnologias utilizadas

* Python
* Pandas
* Matplotlib
* Google Colab

## 📊 Análises realizadas

* Receita por região
* Vendas ao longo do tempo
* Vendas por categoria
* Lucro por região

## 📈 Insights

* Algumas regiões apresentam maior volume de vendas
* O comportamento das vendas varia ao longo do tempo
* Existem diferenças entre categorias de produtos
* O lucro não segue exatamente o mesmo padrão da receita
